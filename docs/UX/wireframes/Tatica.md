# Wireframe — Editor Tático

Fluxo: Dashboard → Editor Tático.

ASCII:

```
+--------------------------------- Editor Tático --------------------------------+
| Formação: [4-4-2]  Ritmo: [Médio]  Linha Def: [Alta]  Pressão: [Média]         |
+--------------------------------------------------------------------------------+
|                                ●        ●                                       |
|                           ●                       ●                             |
|                                                                                 |
|                     ●          ●          ●          ●                          |
|                                                                                 |
|                           ●                       ●                             |
|                                ●        ●                                       |
+--------------------------------------------------------------------------------+
| [Papéis] ST: Poacher | ST: Target Man | MC: BWM | MC: AP | FB: WB(A) ...        |
| [Instruções] Marcação: Individual | Liberdade: Média | Laterais: Ofensivos      |
| [Bolas Paradas] Escanteios, Faltas, Pênaltis                                    |
+--------------------------------------------------------------------------------+
| Salvar Preset | Carregar | Simular Próximo Jogo                                 |
+--------------------------------------------------------------------------------+
```

Requisitos: MX-REQ-010, MX-REQ-011, MX-REQ-012.
